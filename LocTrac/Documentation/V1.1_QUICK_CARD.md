# Version 1.1 - Quick Reference Card

## What's New in 1.1?

### 🎉 Travel History (NEW)
**Location**: Menu → Travel History

**Features**:
- View all your stays in one place
- Filter: All | Other
- Sort: Country | City | Most | Recent  
- Search cities/countries
- Statistics dashboard
- Event details with maps
- Share as text

### 📍 Default Location (IMPROVED)
**Location**: Menu → Manage Locations → Top Section

**Changes**:
- Now integrated in Manage Locations
- No separate menu item
- Benefits display
- Cleaner interface

### 🎨 Color Picker (IMPROVED)
**Location**: Edit Location → Theme Color

**Changes**:
- Native iOS color picker
- Grid, Spectrum, Sliders modes
- Auto-maps to themes

### 🌍 Country Geocoding (NEW)
**Tool**: EventCountryGeocoder.swift

**Features**:
- Parse from city strings
- Detect US states
- Geocode coordinates
- Batch update all events
