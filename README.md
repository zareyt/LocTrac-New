# LocTrac

**Privacy-First Travel Tracking for iOS**

LocTrac is a comprehensive travel tracking application built entirely in SwiftUI. Track your locations, stays, trips, and travel memories with complete privacy—all data stays on your device.

## Features

- 📍 Location Tracking - Record and manage visited locations
- 📅 Stay Management - Track stays with dates, activities, and people
- ✈️ Trip Planning - Organize travels with CO₂ impact tracking
- 📊 Infographics - Visualize travel statistics and patterns
- 🔔 Daily Affirmations - Motivational reminders and notifications
- 🪄 Home Screen Widget - Daily affirmation widget
- 🎨 Custom Colors - Full spectrum location color customization
- 🔒 Privacy First - All data stored locally, no cloud required

## Requirements

- iOS 18.0+ / iPadOS 18.0+
- Xcode 15.0+
- Swift 5.9+
