# Travel Journey Filtering - User Guide

## Overview

The Travel Journey feature now includes powerful filtering options that let you focus on specific trips, activities, or time periods. This guide shows you how to use filters to explore your travel history.

---

## What Can You Filter?

### 📅 Year
See events from a specific year
- Example: "Show only 2024 trips"

### 🏷️ Event Type  
Filter by trip type (Stay, Vacation, Family, Business, Host)
- Example: "Show only vacation trips"

### 📍 Location
Focus on events at a specific location
- Example: "Show only Denver trips"

### 🏃 Activity
See events where you did a specific activity
- Example: "Show all skiing trips"

---

## How to Apply Filters

### Method 1: Quick Filter Menu (Easiest)

1. **Open Travel Journey**
   - Tap the journey button in your app

2. **Tap "Filters" at top**
   - Look for the filter bar with calendar icon

3. **Select filter type**
   - Choose Year, Event Type, Location, or Activity

4. **Pick your filter**
   - Select from available options
   - Journey automatically updates

### Method 2: Settings Menu

1. **Open Travel Journey**

2. **Tap ⋯ icon** (top-right corner)

3. **Scroll to filter section**
   - "Filter by Year"
   - "Filter by Event Type"
   - "Filter by Location"
   - "Filter by Activity"

4. **Choose your filter**
   - Journey resets to start with new filter

---

## Understanding Filter Results

### What You'll See

**Event Count**
- Shows total events matching your filters
- Updates as you add/remove filters
- Displayed as "X events" at top

**Filter Chips**
- Active filters shown as small pills below filter bar
- Each shows the filter name (e.g., "2024", "Vacation")
- Tap the X to remove that filter

**Navigation Title**
- Shows active filters
- Example: "Journey - 2024, Vacation, Denver"
- Returns to "Travel Journey" when no filters

**Red Dot Badge**
- Appears on ⋯ menu icon when filters active
- Quick indicator that filters are applied

---

## Removing Filters

### Remove One Filter

**Tap X on filter chip:**
- Find the chip below filter bar
- Tap the X button on right
- That filter removed
- Other filters stay active

### Remove All Filters

**Use Clear All button:**
1. Tap ⋯ menu icon
2. Scroll to bottom
3. Tap "Clear All Filters" (red button)
4. All filters removed instantly

**Or change to "All":**
- Open any filter picker
- Select "All Years" or "All Types"
- That filter cleared

## Combining Filters

### Why Combine Filters?

Get very specific results:
- "Skiing trips in 2024"
- "Denver vacations"
- "All family events in California"

### How to Combine

1. **Apply first filter**
   - Example: Year = 2024

2. **Apply second filter**
   - Example: Event Type = Vacation

3. **Apply third filter** (optional)
   - Example: Location = Denver

4. **Result:**
   - Only vacation events in Denver in 2024

**Each filter narrows the results further!**

## Troubleshooting

### "No events found"

**What this means:**
- Your filters excluded all events
- No events match current criteria

**How to fix:**
1. Tap "Clear All Filters"
2. Start with broader filter
3. Narrow down gradually

### Filters not showing options

**What this means:**
- That filter type has no events
- Example: No activities assigned to events

**What to do:**
- Only appears if data exists
- Assign activities to events in app
- Then activity filter will appear

### Journey resets when I filter

**This is normal:**
- Journey always starts from beginning with new filters
- Ensures chronological playback
- Shows filtered events in order

### Too many events showing

**How to narrow:**
- Add more filters
- Combine year + type + location
- Each filter reduces results

### Filter chips scrolling off screen

**This is normal:**
- Chips scroll horizontally
- Swipe left to see more
- Or remove some filters

## FAQ

**Q: Do filters save when I close journey?**
A: No, filters reset each time you open journey. This gives you a fresh start.

**Q: Can I save filter combinations?**
A: Not yet. This is a planned future enhancement.

**Q: Why don't I see all filter options?**
A: Filters only show types with actual data. Add more events to see more options.

**Q: Can I filter by date range instead of year?**
A: Not yet. Currently only full year filtering is available.

**Q: Do filters work on maps too?**
A: The map view has year filtering. Journey has more comprehensive filters.

**Q: Can I exclude locations instead of including?**
A: Not yet. Currently filters are inclusive only.

**Q: How many filters can I apply at once?**
A: All 4 types can be active simultaneously (year, type, location, activity).

**Q: Will filters affect my events?**
A: No, filters only affect what's shown in journey. Your events are not modified.
